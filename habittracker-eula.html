Service Description

HabitTracker is a habit tracking application for iOS devices that helps users build and maintain daily habits using the "don't break the chain" methodology. The app provides both free and premium features.

Free Features Include:

  • Track one (1) habit
  • Visual calendar with streak tracking
  • Basic progress statistics
  • Local data storage

Premium Features Include:

  • Unlimited habit tracking
  • Advanced progress analytics
  • Monthly progress reports
  • Priority customer support
